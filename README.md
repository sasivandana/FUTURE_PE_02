# FUTURE_PE_02 – AI UGC Ad Content Generator for D2C Skincare Brands

## Business Chosen

GlowSkin Co. – A D2C skincare brand focused on solving acne, dull skin, and hydration issues using simple, effective products.

---

## Target Audience

Young adults (18–30), especially students and beginners dealing with:

* Acne and breakouts
* Oily or dry skin
* Confusion from using multiple skincare products

---

## Project Objective

To design a structured, reusable prompt engineering system that generates **high-converting UGC-style ad content** for skincare brands.

This system produces:

* Scroll-stopping hooks
* Short-form UGC ad scripts (Reels/Shorts)
* Tone-based variations
* Conversion-focused CTAs
* Platform-ready captions

---

## Problem Identified

Most skincare brands:

* Use overly polished ads that feel inauthentic
* Lack relatable UGC-style storytelling
* Fail to create strong scroll-stopping hooks
* Do not optimize content for short-form platforms
* Miss clear, conversion-driven CTAs

---

## Prompt Engineering Approach

### Hook Generation Framework

Uses curiosity, pain points, and relatability to capture attention.

### UGC Ad Structure

Hook → Problem → Solution → Experience → CTA

### Tone Adaptation System

* Casual (friend-like)
* Emotional (struggle → confidence)
* Informative (simple explanation)

### Conversion Constraints

Each output ensures:

* Clear call-to-action
* Authentic user tone
* Short-form content optimization
